<h1> Segundo Challenge de One | Next Education </h1>

Este repositorio corresponde al Segundo Challenge del programa ONE | Next Education, en el cual se aplican conocimientos fundamentales de desarrollo web para la creación de una solución funcional y estructurada.

Durante este challenge se pusieron en práctica habilidades como:
• Maquetación y estructura de interfaces web
• Uso de HTML, CSS, PHP y JavaScript
• Buenas prácticas de organización de código

El objetivo principal de este proyecto es consolidar los conocimientos adquiridos, fomentar la resolución de problemas y fortalecer la autonomía en el desarrollo de proyectos reales.

Este trabajo forma parte de mi portafolio personal, como evidencia de mi progreso y aprendizaje dentro del programa ONE | Next Education.
